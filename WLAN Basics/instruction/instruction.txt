6. Configure WLAN services on AC

a. Configure AP source interface and AP authentication method
wlan ac source interface vlanif200
 ap-auth-mode no-auth

This should bring up AP online with AC

Verify
[AC6605-wlan-view]display ap all
  All AP information(Normal-1,UnNormal-0):
  ------------------------------------------------------------------------------

  AP    AP                    AP              Profile   AP              AP
                                              /Region
  ID    Type                  MAC             ID        State           Sysname
  ------------------------------------------------------------------------------
  0     AP6010DN-AGN          00e0-fc2c-7c50    0/0     normal          ap-0
  ------------------------------------------------------------------------------



b. Configure WMM profile
[AC6605]wlan 
[AC6605-wlan-view]wmm-profile name wp0 id 0 

c. Configure traffic profile

traffic-profile name tp0 id 0

d. Configure security profile
security-profile name sp0 id 0
  security-policy wpa2
  wpa2 authentication-method psk pass-phrase cipher huawei123 encryption-method ccmp
quit
quit

e. Configure ESS interface 
interface Wlan-Ess0
 port hybrid pvid vlan 10
 port hybrid untagged vlan 10
 port-isolate enable


f. Configure Service Set and bind traffic profile, security profile and service vlan

wlan
service-set name ss0 id 0
  wlan-ess 0
  ssid CAA-LAN
  user-isolate
  traffic-profile id 0
  security-profile id 0
  service-vlan 10

g. Create Radio Profile
radio-profile name rp0 id 0
  wmm-profile id 0

h. bind radio profile and service set to AP
ap 0 radio 0
  radio-profile id 0
  service-set id 0 wlan 1

commit all


Verify that SSID is broadcasting.
